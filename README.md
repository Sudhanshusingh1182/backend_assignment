
# Learning management System (Backend)

Welcome to the backend repository of the Learning Management System (LMS) application. This project provides the core functionality for a platform where students and instructors interact through courses, videos, and progress tracking.


## Features

### General 
#### User authentication
- Login, Sign Up, and Logout functionalities.
- Secure handling of user credentials.

#### Role Based Access Control(RBAC)
- Separate access levels for students and instructors.
- Role-specific operations and privileges.

### Students Features
- View and Study Courses: Students can browse and watch course videos.
- Enroll in Courses: Purchase and enroll in courses of interest.
- Track Progress: Monitor learning progress through the dashboard.

### Instructor Features
- Create and Manage Courses: Instructors can create new courses, upload videos, and provide course descriptions.
- Student Tracking: Track student progress in enrolled courses.
- Manage Enrollments: View the list of students enrolled in specific courses.





## Installation

1. Clone the repository

```bash
  git clone https://github.com/Sudhanshusingh1182/backend_assignment/

```

2. Navigate to the project directory

```bash
  cd lms-backend
```

3. Install Dependencies
```bash
  npm install
```
4. Create a .env file and set up the environment variables.

5. Start the server
```bash
  npm start
```

## Tech Stack

**Node.js:** Backend runtime environment

**Express.js:** Framework for building REST APIs.

**MongoDB:** Database for storing application data.

**JWT:** Authentication and Authorization
